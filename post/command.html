      Εντολή Περιγραφή
      print('Κείμενο') Εμφανίζει ένα κείμενο στην οθόνη
      if (συνθήκη): Ελέγχει μια συνθήκη και εκτελεί κώδικα αν η συνθήκη είναι αληθής.
      elif (συνθήκη): Εκτελεί κώδικα αν η συνθήκη στην if δεν είναι αληθής αλλα η συνθήκη στην elif είναι αληθής.
      else Εκτελεί κώδικα αν η συνθήκη στην if δεν είναι αληθής.ς
      for Επαναλαμβάνει έναν κομμάτι κώδικα για έναν ορισμένο αριθμό επαναλήψεων.
      while (συνθήκη): Επαναλαμβάνει έναν κομμάτι κώδικα όσο μια συνθήκη είναι αληθής.
      def function_name(arguments): Ορισμός συνάρτησης με ορίσματα
      return value Επιστρέφει μια τιμή από μια συνάρτηση.
      input Διαβάζει τιμές από τον χρήστη από την κονσόλα.
      break Τερματίζει την εκτέλεση ενός βρόχου.
      continue ΤΠαρακάμπτει τον υπόλοιπο κώδικα ενός βρόχου και συνεχίζει με την επόμενη επανάληψη.
