    Python3 | Terminal

    Web Server

    Δημιουργήστε έναν HTTP Server

    python3 -m http.server 8000

    Για να εκκινήσετε τον HTTP server στην διεύθυνση IP του υπολογιστή σας, μπορείτε να καθορίσετε την επιθυμητή διεύθυνση IP ως όρισμα της εντολής.

    python3 -m http.server --bind 192.168.1.1

    Convert

    Μετατροπή από δεκαδικό σε άλλα αριθμητικά συστήματα

    python3 -c 'print(bin(42), oct(42), hex(42))'

    Μετατροπή από δυαδικό σε δεκαδικό

    python3 -c 'print(int("1101", 2))'

    Μετατροπή από άλλα αριθμητικά συστήματα σε δεκαδικό

    python3 -c 'print(int("1010", 2), int("52", 8), int("1A", 16))'

    Μετατροπή αρχείων

    XML
    Το module xml.etree.ElementTree μπορεί να χρησιμοποιηθεί για τη μετατροπή αρχείων XML σε Python δομές δεδομένων και αντίστροφα.

    python -m xml.etree.ElementTree my_xml_file.xml

    Μετατροπή αρχείων Markdown σε HTML

    python -m markdown2 input.md > output.html
